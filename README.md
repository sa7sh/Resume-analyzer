# AI Resume Analyzer

A full-stack AI-powered application that analyzes resumes using ATS scoring, keyword matching, and AI-driven feedback.

## 🔧 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- AI: Google Gemini API
- Deployment: Vercel (frontend), Render (backend)

## 🚀 Features
- Resume parsing (PDF)
- ATS score calculation
- Keyword gap analysis
- AI-generated feedback
- Role-based scoring system

## 🌐 Live Demo
Frontend: https://resume-analyzer-delta-weld.vercel.app
Backend: https://resume-analyzer-api-8ocp.onrender.com

## ⚙️ Setup

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev  

## 🔐 Environment Variables
GEMINI_API_KEY=your_api_key
